Bem Vindo

A Microsoft anunciou oficialmente em junho o Windows 11, novo sistema operacional da empresa para PCs. Embora a versão final esteja planejada para o segundo semestre, usuários interessados em experimentar as novidades da plataforma já podem instalar uma prévia do Windows 11 – confira abaixo os requisitos técnicos e o que você precisa fazer para começar a testar o novo sistema operacional. No fim de junho, a Microsoft começou a disponibilizar as prévias do Windows 11. Com recursos limitados – sem, por exemplo, integração com Teams ou apps do Android -, essas versões são voltadas para entusiastas do sistema operacional. Como não é uma versão finalizada da plataforma, a prévia do sistema pode apresentar bugs e problemas de compatibilidade em algumas máquinas – o que é normal no processo de desenvolvimento.
Para fazer download do Windows 11 é necessário se cadastrar no programa Windows Insider (o que é gratuito). Depois, em um PC com Windows 10, siga os seguintes passos: Vá até Configurações Depois, entre em Atualização e Segurança Agora, clique em Programa Windows Insider Lá é necessário ativar o canal “Dev”, que contém software ainda em desenvolvimento Não custa lembrar que esta prévia é uma versão muito preliminar do sistema, que não deve ser instalada em máquinas que desempenham funções críticas ou das quais você depende no dia a dia para trabalhar ou estudar. A melhor forma de conhecer as novidades sem riscos é instalar o sistema em um PC “cobaia” – como um notebook antigo, por exemplo.
